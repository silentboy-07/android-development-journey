# Android Development Journey 🚀

## 📅 Day 7 - Cards in Jetpack Compose

## ✅ What I Did

* Created a new project **Cards**
* Practiced creating cards in Jetpack Compose
* Built simple and profile-style card UI designs

## 📚 What I Learned

### 🔹 Card Composable

* Used `Card()` to create container UI components
* Learned properties:

  * `shape`
  * `colors`
  * `elevation`
  * `border`
  * `modifier`

### 🔹 Card Styling

* Used `RoundedCornerShape()` for rounded corners
* Added shadows using `cardElevation()`
* Added borders using `BorderStroke()`
* Customized background and text colors

### 🔹 Layout Inside Card

* Used `Row()` for horizontal layout
* Used `Column()` for vertical content
* Used `Box()` for centered icon section
* Used `Spacer()` for spacing

### 🔹 Profile Card UI

* Created a profile-style card with:

  * Person icon
  * Name text
  * Description text

### 🔹 UI Design Concepts

* Content alignment
* Padding and spacing
* Card-based modern UI structure

## 📌 Outcome

* Built a simple text card
* Built a professional profile card UI
* Improved understanding of reusable UI containers
* Learned how cards are used in real Android apps
