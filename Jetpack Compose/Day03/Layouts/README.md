# Android Development Journey 🚀

## 📅 Day 3 - Layouts in Jetpack Compose

## ✅ What I Did

* Created a new project **LayoutApp**
* Practiced different layout composables in Jetpack Compose
* Built simple UI using layout containers

## 📚 What I Learned

### 🔹 Row Layout

* Used `Row()` to arrange items horizontally
* Learned properties:

  * `horizontalArrangement`
  * `verticalAlignment`
  * spacing between items

### 🔹 Column Layout

* Used `Column()` to arrange items vertically
* Learned properties:

  * `verticalArrangement`
  * `horizontalAlignment`
  * spacing between items

### 🔹 Box Layout

* Used `Box()` to stack items on top of each other
* Learned properties:

  * `contentAlignment`
  * layering UI components
  * alignment inside box

### 🔹 Modifier Usage

* Practiced:

  * `fillMaxSize()`
  * `fillMaxWidth()`
  * `fillMaxHeight()`

## 📌 Outcome

* Built layouts using Row, Column, and Box
* Improved understanding of UI alignment and spacing
* Learned how to structure screens in Compose.
