# Android Development Journey 🚀

## 📅 Day 2 - Basic Composables (Text & Input Fields)

### ✅ What I Did

* Created a new project **BasicComposable**
* Explored basic UI components using Jetpack Compose
* Implemented text display and input fields

---

### 📚 What I Learned

#### 🔹 Text

* Used `Text()` to display text on screen
* Customized:

  * `color`
  * `fontSize`
  * text styling

---

#### 🔹 TextField

* Used `TextField()` for user input
* Learned properties:

  * `label` → shows title inside field
  * `placeholder` → hint text
  * `leadingIcon` → icon at start
  * `trailingIcon` → icon at end
  * `singleLine` → restrict input to one line

---

#### 🔹 OutlinedTextField

* Used `OutlinedTextField()` for bordered input fields
* Learned properties:

  * `label`
  * `placeholder`
  * `leadingIcon`
  * `trailingIcon`
  * `singleLine`
  * `shape` → customize border shape
  * `colors` → change border and text colors
