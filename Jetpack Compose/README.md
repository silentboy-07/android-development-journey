# 📱 Jetpack Compose Journey — Day 1 to 15

> A structured 15-day deep dive into modern Android UI development using Jetpack Compose.

---

## 🗂️ Overview

This section of the repository documents my hands-on learning journey with **Jetpack Compose** — Google's modern, declarative UI toolkit for Android. Each day focuses on a specific concept, building progressively toward a solid foundation in real-world Android development.

---

## 📚 Day-by-Day Breakdown

| Day | Topic | Concepts Covered |
|-----|-------|-----------------|
| 01 | **Getting Started** | First Compose app, project structure |
| 02 | **Basic Composables** | `Text`, `Button`, `Image` |
| 03 | **Layouts** | `Column`, `Row`, `Box`, Arrangement & Alignment |
| 04 | **Modifiers** | Padding, Size, Background, modifier chaining |
| 05 | **Images & Icons** | Displaying images, Material icons |
| 06 | **Buttons** | Customization, styling, click handling |
| 07 | **Cards** | Card UI design, elevation & shapes |
| 08 | **Dividers** | UI separation, spacing improvements |
| 09 | **State Management** | `remember`, `mutableStateOf`, `rememberSaveable`, state hoisting |
| 10 | **AlertDialog** | Creating dialogs, handling user actions |
| 11 | **Navigation** | Screen navigation, type-safe routes |
| 12 | **Lists** | `LazyColumn`, `LazyRow`, nested lists, `items`, `itemsIndexed` |
| 13 | **Scaffold** | `TopAppBar`, `BottomAppBar`, `FloatingActionButton`, screen structure |
| 14 | **Bottom Navigation Bar** | `NavigationBar`, multi-screen navigation |
| 15 | **Toast & Snackbar** | Showing toasts, Snackbar with actions |
