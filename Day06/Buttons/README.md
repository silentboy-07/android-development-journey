# Android Development Journey 🚀

## 📅 Day 6 - Buttons in Jetpack Compose

## ✅ What I Did

* Created a new project **Buttons**
* Practiced different types of buttons in Jetpack Compose
* Built interactive UI with click actions and form validation

## 📚 What I Learned

### 🔹 Button

* Used `Button()` for normal clickable buttons
* Learned properties:

  * `onClick`
  * `shape`
  * `colors`
  * `border`
  * `elevation`
  * `enabled`

### 🔹 OutlinedButton

* Used `OutlinedButton()` for bordered button design
* Customized colors and full width layout

### 🔹 TextButton

* Used `TextButton()` for lightweight text-based actions
* Also made `Text()` clickable using `clickable()`

### 🔹 IconButton

* Used `IconButton()` with Material Icons
* Practiced clickable icons

### 🔹 Icons Inside Button

* Added `Row()` inside `Button()`
* Combined:

  * `Icon()`
  * `Spacer()`
  * `Text()`

### 🔹 Toast Message

* Used `Toast.makeText()` to show messages on button click

### 🔹 Login Form Practice

* Created a simple login screen using:

  * `OutlinedTextField()`
  * Username field
  * Email field
  * Login button

### 🔹 Validation Logic

* Used `enabled` property to activate button only when:

  * Username is not empty
  * Email is not empty

## 📌 Outcome

* Built multiple button examples
* Learned click handling in Compose
* Created a basic login UI with validation
* Improved UI interaction skills
