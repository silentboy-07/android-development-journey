# Android Development Journey 🚀

## 📅 Day 4 - Modifiers in Jetpack Compose

## ✅ What I Did

* Created a new project **Modifiers**
* Practiced different modifiers in Jetpack Compose
* Designed UI using Row, Column, and Box layouts with styling

## 📚 What I Learned

### 🔹 Size & Layout Modifiers

* Used `fillMaxWidth()` to occupy full width
* Used `size()` to define fixed width and height
* Applied `padding()` for inner spacing

### 🔹 Styling Modifiers

* Used `background()` to apply background colors
* Used `border()` to add borders
* Used `RoundedCornerShape()` for rounded corners

### 🔹 Interaction Modifier

* Used `clickable()` to make composables clickable

### 🔹 Alignment & Arrangement

* Used `horizontalAlignment`
* Used `contentAlignment`
* Used `Arrangement.SpaceAround`
* Used `Arrangement.spacedBy()`

## 📌 Outcome

* Built a structured UI using:

  * `Column`
  * `Row`
  * `Box`
* Improved understanding of layout styling and spacing
* Learned how modifiers control UI appearance and behavior
