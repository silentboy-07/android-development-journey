# Android Development Journey 🚀

## 📅 Day 13 - Scaffold in Jetpack Compose

## ✅ What I Did

* Created a new project **Scaffold**
* Practiced building complete app layout using Jetpack Compose
* Implemented Top App Bar, Bottom App Bar, Floating Action Button, and content section

## 📚 What I Learned

### 🔹 Scaffold Layout

* Used `Scaffold()` as the main screen structure
* Learned how to organize app UI into separate sections:

  * Top Bar
  * Bottom Bar
  * Floating Action Button
  * Main Content Area

### 🔹 Top App Bar

* Used `TopAppBar()`
* Added:

  * Title text
  * Navigation menu icon
  * Search icon
  * More options icon
* Customized colors using `TopAppBarDefaults.topAppBarColors()`

### 🔹 Bottom App Bar

* Used `BottomAppBar()`
* Added navigation icons:

  * Home
  * Search
  * Notifications
  * Profile

### 🔹 Floating Action Button

* Used `FloatingActionButton()`
* Added plus icon for quick action button

### 🔹 Content Area

* Used `LazyColumn()` inside Scaffold body
* Displayed multiple card items in scrollable list

### 🔹 Card UI Design

* Built reusable card layout using:

  * `Card()`
  * `Row()`
  * `Column()`
  * `Box()`
  * `Icon()`
  * `Text()`

### 🔹 Layout Handling

* Used `innerPadding` from Scaffold
* Managed spacing and alignment properly

## 📌 Outcome

* Built a professional app screen layout
* Learned structure used in real Android apps
* Improved UI organization skills
* Combined multiple Compose components in one screen
