# Android Development Journey 🚀

## 📅 Day 5 - Displaying Images & Icons in Jetpack Compose

## ✅ What I Did

* Created a new project **Images**
* Practiced displaying images and icons using Jetpack Compose
* Used both pixel images and vector images in UI design

## 📚 What I Learned

### 🔹 Image Composable

* Used `Image()` to display images on screen
* Learned properties:

  * `painterResource()`
  * `contentDescription`
  * `contentScale`
  * `size()`
  * `padding()`
  * `clip()`

### 🔹 Icon Composable

* Used `Icon()` to display icons
* Learned properties:

  * `imageVector`
  * `contentDescription`
  * `tint`
  * `size()`

### 🔹 Types of Images

#### Pixel Images

* Used for photos and detailed images
* Examples:

  * `.png`
  * `.jpg`
  * `.jpeg`
  * `.webp`

#### Vector Images

* Used for icons and scalable graphics
* Quality remains clear at any size
* Examples:

  * XML Vector Drawable
  * Material Icons

## 📌 Outcome

* Displayed a rounded image using `Image()`
* Displayed a person icon using `Icon()`
* Understood difference between pixel and vector images
* Improved UI design skills with images and icons
