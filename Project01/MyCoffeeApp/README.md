# ☕ My Coffee App (Jetpack Compose Practice)

A modern Coffee App UI built using **Jetpack Compose** as part of my Android Development Journey 🚀  
This project focuses on practicing core Jetpack Compose concepts and building real-world UI screens.

---

## 📱 Features

- ✨ Welcome Screen
- 🏠 Home Screen (Coffee listing UI)
- 📄 Detail Screen
- 🛒 Cart Screen
- 👤 Profile Screen
- 🎨 Clean and modern UI using Material 3

---

## 🧠 Purpose of This Project

This project is built to:

- Practice Jetpack Compose fundamentals
- Understand UI design and layout structuring
- Build a multi-screen Android app
- Strengthen reusable component design

> ⚠️ Note: This is a **frontend-only project** (No backend integration yet)

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
